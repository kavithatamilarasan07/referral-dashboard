# 📊 Referral Dashboard

A simple Referral Dashboard web application built using React.js to manage and track user referrals and profits.

---

## 🚀 Features

- Add new referral details
- Track referral profit
- View list of all referrals
- Simple and clean UI
- Responsive design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
